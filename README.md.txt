# Study Quest - Learning Gamification Platform

## Problem Statement
Students across Latvia struggle with motivation and engagement in education. Traditional learning methods fail to inspire, leading to:
- 85% of students report lack of motivation
- 70% suffer from procrastination
- Decreasing academic performance
- High levels of learning burnout

## Our Solution
**Study Quest** transforms learning into an engaging adventure through gamification. We make education fun, rewarding, and effective.

## Key Features
- 🎯 **Quest System** - Turn homework into exciting challenges
- ⭐ **XP & Level Progression** - Visual growth tracking
- ⏱️ **Built-in Pomodoro Timer** - Focus management
- 🏆 **Achievements System** - Milestone recognition
- 🛒 **Reward Shop** - Spend earned coins on customization
- 📊 **Progress Visualization** - See your learning journey

## Technology Stack
- HTML5, CSS3, JavaScript (ES6+)
- Local Storage for data persistence
- Responsive design
- Gamification mechanics

## How to Use
1. Clone repository
2. Open `index.html` in browser
3. Start completing quests and level up!
4. Use Pomodoro timer for focused sessions
